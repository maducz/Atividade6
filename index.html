6. O que é A.P.I?

Interface de Programação de Aplicações é um conjunto de regras

8. O que significa "responsividade" em páginas WEB?

A responsividade de uma página WEB refere-se à capacidade de um site se adaptar
e oferecer uma experiência de usuário consistente em diferentes despositivos e aparekhos de tela.

9. Qual o papel do U.X. (User Experience)?

O papel do U.X é garantir que os usuários tenham uma eperiência positiva e satisfatória com o site,
assim como o feedback.

10. O que é um wireframe em páginas WEB?

O wireframe são grades para organizar um site.
